## About

Hack 'n' Slash is a community-driven, open-source action game where players battle waves of enemies using fast-paced melee combat. The goal is to build a fun, extensible game that anyone can play, modify, and contribute to.

---

## Features

- ⚔️ Fast-paced hack and slash combat
- 🗺️ Multiple levels and enemy types
- 🎮 Gamepad and keyboard/mouse support
- 🖥️ Cross-platform: Windows, macOS, and Linux
- 🔓 Fully open source

## Getting Started

### Prerequisites

- [VS Code](https://code.visualstudio.com/) (recommended editor)
- A compatible game engine or runtime (e.g., [Godot Engine](https://godotengine.org/) 4.x)
- Git

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/tylertech234/hacknslash.git
   cd hacknslash
   ```

2. **Open in VS Code:**
   ```bash
   code .
   ```

3. **Open the project in your chosen game engine** (e.g., import the project folder into Godot Engine).

### Running the Game

- Open the project in your game engine and press the **Play** button, or
- Use the engine's export feature to build a standalone executable for your platform.
